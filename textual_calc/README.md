# App
Dummy app made with textual

## Requirements
```console
$ python -m venv venv
$ venv\Scripts\activate 
$ pip install textual
```

## How to Run
```console
$ python app.py
```


## How to create an executable

```console
pip install pyinstaller
```

```console
pyinstaller app.py 
```
